# Química Revisada — Revisão de Química para o ENEM

Plataforma gratuita de revisão de Química para o ENEM com vídeos, conteúdos essenciais e simulados interativos.

## 🚀 Deploy no GitHub Pages

1. Crie um repositório no [GitHub](https://github.com/new) com o nome `quimicarevisada`
2. Envie os arquivos:
   ```bash
   cd quimicarevisada
   git init
   git add .
   git commit -m "Primeiro commit"
   git branch -M main
   git remote add origin https://github.com/SEU-USUARIO/quimicarevisada.git
   git push -u origin main
   ```
3. Ative o GitHub Pages: **Settings → Pages → Branch: main → Save**
4. Acesse: `https://SEU-USUARIO.github.io/quimicarevisada/`

## ✏️ Personalização

- **Vídeos**: Substitua os placeholders por embeds do YouTube na seção de vídeos
- **Questões**: Adicione mais questões em `questions.js`
- **Contato**: Atualize email e redes sociais no footer
